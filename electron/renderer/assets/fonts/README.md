# Visual test font

`AutoVPNVisualTest-NotoSansSC.woff2` is a test-only subset of Noto Sans SC. It contains the glyphs rendered by the H5 visual fixtures and is activated only when the test harness sets `data-visual-test` on the root element. The subset keeps text metrics stable across macOS and Linux without changing the production typography.

The source font is distributed under the SIL Open Font License; see `OFL.txt`.
